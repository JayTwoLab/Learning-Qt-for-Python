This repository contains a collection of simple and educational **PySide2 GUI programming examples**.  
Each script demonstrates a specific Qt widget, layout, or feature.  
They are based on sample codes from the *"PySide GUI Application Development (2013)"* series.

## File Overview

### 1. 01.creating-the-main-window.py
A minimal example showing how to create a basic `QMainWindow`, set its title, size, and launch it inside a Qt application loop.

### 2. 02.Status-bar.py
Adds a status bar to the main window and displays temporary messages.

### 3. 03.Working-progressbar.py
Implements a status bar with a label and progress bar. A simulated download loop updates the progress bar.

### 4. 04.Simple-text-editor.py
A basic text editor implemented using `QTextEdit` and `QTextDocument`.

### 5. 05.Adding-Menubar.py
Introduces menu bars, actions, and keyboard shortcuts.

### 6. 06.Toolbar.py
Builds on the previous example by adding a toolbar.

### 7. 07.Layout.py (Horizontal Layout)
Demonstrates a horizontal layout (`QHBoxLayout`) with stretchable spacing.

### 8. 08.Layout.py (Vertical Layout)
Shows a vertical layout using `QVBoxLayout`.

### 9. 09.Layout.py (Grid Layout)
Uses a grid layout to arrange widgets in rows and columns.

### 10. 10.Widget.py (Form Layout)
Shows a form layout with labels and input fields.

## Running the Examples

Run any example with:

```
python <filename>.py
```

Requires:

- Python 3.x  
- PySide2 (`pip install PySide2`)
