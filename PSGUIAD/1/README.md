# PySide2 Example Scripts

This repository contains several simple Python examples demonstrating
how to build GUI applications using **PySide2** (Qt for Python). Each
example highlights different aspects of Qt widgets, signals/slots, error
handling, or environment checks.

## File Overview

### 1. hello_world.py

A minimal PySide2 application that displays a window with a single
label.

### 2. hello_world2.py

An interactive example using widgets, layouts, and signals.

### 3. hello-exception.py

Demonstrates exception handling inside a Qt application.

### 4. version.py

Prints the installed PySide2 version.

## How to Run

``` bash
python hello_world.py
python hello_world2.py
python hello-exception.py
python version.py
```

## Requirements

-   Python 3.7+
-   PySide2
